## World Explorer GQL
A Single-page-app that fetches information about countries from a GraphQl endpoint & listing results.

<br />

## See live: https://world-explorer-gql.netlify.app/

<br />

## Quick Start

- npm install
- npm run dev
- open `http://localhost:3000`
